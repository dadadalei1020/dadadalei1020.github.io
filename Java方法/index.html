Java方法

在java中，方法就是用来完成解决某件事情或实现某个功能的办法。方法实现的过程中，会包含很多条语句用于完成某些有意义的功能——通常是处理文本，控制输入或计算数值。我们可以通过在程序代码中引用方法名称和所需的参数，实现在该程序中执行（或称调用）该方法。方法，一般都有一个返回值，用来作为事情的处理结果。

main就是一个方法.

作用:作为的程序入口.

方法的定义格式:

修饰符 返回值 方法名(参数列表,每个参数之间用逗号隔开

eg:数据类型1 参数名1,数据类型2 参数名2){

方法体;

return 数据;(又返回值就写return,没有返回值可以省略)

}

修饰符 :修饰方法的作用范围及调用的方式,public 和 static

public 就是公共的公开的.static静态的 修饰方法的调用者

返回值类型:方法调用完之后,要给调用者返回的数据的格式,

如果没有返回值:void

方法名:给方法取个名字,方便调用者调用,命名方式遵循标识符的规则

方法名要见名知意,使用小驼峰命名法

参数列表:调用方法时给内部传入数据用的

定义方法的注意事项:

1.不能再别的方法内部定义方法,把写的除了类括号内的,任何一个大括号内

2.方法定义完之后,如果不调用,方法自己是不会执行的

调用:格式:方法名(参数(没有则不写));

方法的好处:

1.可以实现功能的封装(隐藏)起来,只对对外公开方法名(接口)

2.简化代码量,提示开发效率

3.方法是能复用的

根据方法是否有返回值是否有参数,将方法分为四类:
1.无参数无返回值

无参无返的方法格式: 修饰符 void 方法名(){ 方法体; }

无参无返方法的调用:方法名();

2.无参数有返回值

无参有返的格式: 修饰符 数据类型 方法名(){ 方法体; return 数据类型;

return返回的数据类型一定要和方法名前面数据类型一致 }

无参有返方法的调用: 数据类型 变量名 = 方法名();

注意:定义变量接收返回值数据类型要和方法定义时数据类型一致

3.有参数无返回值

有参无返的方法格式: 修饰符 void 方法名(数据类型1 参数名,数据类型2 参数名…){ 方法名; }

参数的数据类型如何确认:根据方法内部需要的方法类型来设置

带有参数方法如何调用:一个方法如果有参数,调用必须传入参数,待入的参数数据类型,参数个数及参数顺序一定 要方法定义时一致

4.有参数有返回值

有参有返的方法格式: 修饰符 数据类型 方法名(参数类型1 参数名1,参数类型2 参数名2..)

{ 方法体; rutern 数据类型;}

注意:在定义方法的时候,如何确定返回值数据类型,一定要根据具体需求设定

定义方法时候的参数称之为形式参数

调用方法传入的参数称之为实际参数

传递方法:
值传递:

就是把一个变量中的值复制一份,给了另一个变量,两个变量不指向同一块内存空间,两个变量是独立的

址传递:

就是把一个变量中的地址复制一份,给了另一个变量,两个变量指向同一块内存空间,两个变量一样的

注意事项:定义方法的时候,不要让方法重复
